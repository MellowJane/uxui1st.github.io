디지털컨버전스(UI/UX엔지니어링,프론트앤드)양성과정 10회차 작품발표

훈련기간 : 2017.03.13 ~ 2017.08.02

  1. 김웅기 dot 홈페이지 리뉴얼 닷은 시각장애인용 점자 스마트워치를 제작하는 스타트업입니다. 기존의 홈페이지를 리뉴얼하여 기업형 디자인으로 만들었으며, 모든 페이지는 RWD로 제작되었고, php기반으로 제작하였습니다.
  2. 박재서 (주)티비에이치글로벌 - 패션기업 PC용 웹 리뉴얼 기존의 웹사이트는 일부 상황에서 네비게이션이 화면에서 사라진는 문제가 있어, 리뉴얼시에는 고정 네비게이션을 사용하는 점에 기능적으로 중점을 두었습니다. 내용적으로는 회사의 브랜드를 보여주는 데 초점을 맞추었고, 시각적으로는 회사의 대표 브랜드 'BASIC HOUSE'가 형성한 깨끗한 이미지를 반영하고자 하였습니다.
  3. 김웅기 super tour korea 홈페이지 리뉴얼 super tour korea는 외국인을 대상으로 서울 자전거 여행 상품을 판매하는 회사입니다. 페이지는 데스크탑 전용으로 제작되었으며, 기존의 복잡한 구조와 조금 낡은 듯한 디자인을 개선하였습니다. 여행의 밝은 느낌을 살리기 위해 다양한 애니메이션 효과를 사용하였습니다..
  4. 문희정 '파리바게트'의 사이트 리뉴얼 대형 베이커리 프랜차이즈인 '파리바게트'의 사이트 리뉴얼로 회사가 가지고 있는 아이덴티티를 부각시키기 위해 기존의 컬러와 다른 컬러들을 배합하여 디자인 하였습니다. 구조적으로 컨텐츠들의 영역을 명확하게 구분하였으며 사용자들이 접근하기 편하게끔 접근성 검사와 메뉴와 전체적인 레이아웃을 단순화하여 제작하였습니다.
  5. 김웅기 Magazine B 모바일 페이지 제작. 좋은 기업을 선정하여 기업을 주제로 잡지를 발간하는 Magazine B의 모바일 전용 페이지를 만들었습니다. 기존에 모바일용 페이지가 존재하지 않았기에, 색이나 요소의 배치는 기존의 홈페이지와 최대한 느낌이 비슷하게 맞추었습니다.
